发表于 2018-01-10 | 分类于 React

1. State 属性

State 属性：

  1. state 对于模块属于自身属性
  2. 初始化: 
    this.state = {username:"Parry"};
  3. 初始化可以放置在构造函数 constructor 里
  4. 修改 state: 
    this.setState{{username:"IMOOC"}};
  5. state 的作用域只属于当前的类，不污染其他模块
